1. Chuẩn bị danh sách

Tải template Excel, điền danh sách rồi kéo-thả file vào ô dưới. Cột CCCD và Số điện thoại cần để dạng Text để không mất số 0 đầu.

📄 Kéo-thả file .xlsx vào đây hoặc bấm để chọn file
